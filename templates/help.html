Help Center

Getting Started

What is Sam.AI?

Sam.AI is your personal AI assistant that helps you with various tasks, from answering questions to helping with research, writing, and more.

Basic Commands

You can interact with Sam.AI using natural language. Here are some examples:

  • "Tell me about quantum computing"
  • "Write a blog post about sustainable energy"
  • "Summarize this article: [paste URL]"
  • "What's the weather in New York?"

Advanced Features

Web Browsing

Sam.AI can browse the web for you. Try commands like:

  • "Search for latest AI news"
  • "Find tutorials on machine learning"
  • "Look up the recipe for chocolate chip cookies"

Content Generation

Sam.AI can help you create various types of content:

  • "Write an email to my boss requesting time off"
  • "Create a marketing plan for a new product"
  • "Generate a list of ideas for my YouTube channel"

Account & Settings

Managing Your Account

You can manage your account settings by clicking on the Settings icon in the sidebar.

API Integration

To use external APIs with Sam.AI, go to Settings and add your API keys in the API Keys section.
